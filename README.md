# Data Governance and Security Dashboard

## 📌 Overview
This project was developed as part of my internship at Presidency University, Bengaluru.  
It focuses on building **dual interactive dashboards** using Tableau to visualize:

- **Environmental, Social, and Governance (ESG) indicators** (World Bank & Protected Planet API datasets)  
- **Enterprise Data Governance metrics** (simulated organizational data)

The dashboards provide actionable insights into sustainability performance, data quality, privacy compliance, and security incidents.

---

## 🎯 Objectives
- Develop a **dual-dashboard system** (ESG + Governance).  
- Integrate and transform **multi-source datasets** (World Bank, Protected Planet API, simulated enterprise data).  
- Design **interactive visualizations** with Tableau (maps, bar charts, line graphs, pie charts).  
- Monitor governance metrics: data quality, access requests, privacy compliance, security incidents, audit outcomes.  
- Promote **ethical data stewardship** and transparency.  

---

## 🛠️ Tools & Technologies
- **Tableau Desktop** – dashboard design & visualization  
- **SQL** – querying and calculated metrics  
- **Microsoft Excel** – preprocessing and data cleaning  
- **WEKA** – attribute analysis and preprocessing  
- **World Bank Open Data** – ESG indicators  
- **Protected Planet API** – environmental compliance data  

---

## 📊 Data Sources
### ESG Indicators
- Net Migration (SM.POP.NETM)  
- Fertility Rate (SP.DYN.TFRT.IN)  
- Renewable Electricity Usage (EG.ELC.RNEW.ZS)  
- Internet Penetration (IT.NET.USER.ZS)  
- Protected Area Coverage (ER.PTD.TOTL.ZS)  

### Governance Metrics (Simulated)
- Data Quality (accuracy, completeness, consistency)  
- Access Requests (approved/denied/pending)  
- Privacy Compliance (exposure, remediation rate)  
- Security Incidents (type, detection/response time)  
- Audit Outcomes (status, non-compliance count)  

---

## 📐 Methodology
1. **Data Collection** – ESG datasets + simulated governance data  
2. **Data Cleaning & Transformation** – Excel + SQL  
3. **Dashboard Design** – Tableau dashboards (ESG + Governance)  
4. **Interactivity** – filters, drill-downs, tooltips, legends  
5. **Testing & Insights** – validation of metrics and usability  

---

## 📈 Key Insights
- **ESG Dashboard**:  
  - Norway & Iceland lead in renewable energy adoption (>90%).  
  - Fertility rates highest in Sub-Saharan Africa, lowest in Europe.  
  - Net migration highlights geopolitical shifts (Germany, Canada positive; Syria, Venezuela negative).  

- **Governance Dashboard**:  
  - Accuracy & completeness declined steadily in 2024.  
  - Operations dept. had lowest privacy compliance (0.36).  
  - Unauthorized access was the most frequent incident type.  
  - Out of 24 audits, only 8 passed (SOX & GDPR failures most common).  

---

## 📌 Limitations
- Governance data is **simulated**, not live enterprise data.  
- Dashboards are **static** (no real-time streaming).  
- Advanced analytics (predictive modeling, anomaly detection) not included.
